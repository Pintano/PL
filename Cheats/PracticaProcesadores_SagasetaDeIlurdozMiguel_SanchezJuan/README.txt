Trabajo realizado por Juan Sánchez Albardíaz y Miguel Sagaseta de Ilúrdoz.

Objetivos alcanzados (desde rutinas semánticas en adelante):
	-> Asignación de variables
	-> Operaciones con expresiones aritméticas
	-> Operaciones con expresiones booleanas
	-> Gestión del control de flujo

Características específicas:
	-> Usamos Fortram para gestionar las variables Booleanas, con lo que cualquier variable booleana debe empezar por B y cualquiera no booleana no puede empezar por B
	-> Después de falgoritmo añadimos un punto, de esta forma: falgoritmo.

Pasos para la corrección:
	-> Compilar el parser usando la shell llamada "compilar.sh"
	-> Ejecutar el fichero "a.out" seguido de la dirección de un texto de ejemplo, de esta forma: ./a.out ejemplos/ejemplo2.txt
	-> Esto dará como resultado dos ficheros .txt, la tabla de símbolos y la de cuádruplas respectivamente, que pueden distinguirse fácilmente por sus nombres.
